## UdaciCards

A project to create decks and cards in React Native

### Run mobile application
- npm install or yarn install
- node server or yarn start

## Folder structure
```bash
├── README.md - This file.
├── actions # Actions
├── components
  ├── navigators
├── reducers # Reducers
├── utils # API, helpers, styles
App.js # Main file 
...
```
